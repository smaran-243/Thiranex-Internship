Things I've built

A few projects that show how I think about structure, performance, and access — not just visual polish.

Trailhead — hiking log app

An offline-capable route tracker. The map itself is mouse-driven, but every feature — searching trails, saving a route, reviewing elevation data — has a fully keyboard-operable equivalent with announced state changes.

  • React
  • Leaflet
  • IndexedDB
  • ARIA live regions

Ledgerly — budgeting dashboard

Charts in this dashboard are backed by real data tables, not canvas-only graphics, so screen-reader users get the same numbers sighted users see in the visualization. Tested with NVDA and VoiceOver, not just automated scanners.

  • Vue
  • D3
  • PostgreSQL
  • WCAG 2.2 AA

Signpost — accessible design system

A component library built so accessibility is the default, not an add-on: every input ships with a label, every interactive icon ships with a name, and focus order is part of the component contract.
